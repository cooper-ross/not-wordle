# Not Wordle

It's a wordle-like game, but not. This was made for fun during [SCNC 2026](https://www.skillscompetencescanada.com/en/event/skills-canada-national-competition-2026/), based on the design guide for the game competitors were given this year. It's pretty cobbled together, but it has confetti! You can change the length of words in settings, and although this one doesn't have a wordlist or any kind of verification, it does the letter highlighting correctly, and the [demo](https://cooperr.dev/projects/wordle/) has both a setting to change the # of rows/cols and a true word list. Feel free to steal the code! <br>
